# AI Meal Planning Backend

Backend service for the AI Meal Planning application built with Node.js and Flask.

## Quick Start

```bash
npm install
cp .env.example .env
npm run start
```

## Project Structure

```
app/
  controllers/   # API route handlers
  models/        # Data models
  services/      # Business logic
  routes/        # API routes
```

## API Documentation

- Swagger UI: http://localhost:8000/docs

## Running Tests

```bash
npm test
```
